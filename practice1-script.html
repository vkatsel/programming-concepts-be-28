Пастка 1: “Пастка математика” (Проблема input() та типів)

  • Що робить функція input()?.
  • Що виведе програма?
a = input("введіть 5")
b = input("введіть 10")
print(a + b)
  • Чому так сталося? (Тому що input() завжди повертає рядок (str)“. Python не вміє додавати числа до тексту, тому він”склеює” (конкатенує) два рядки.)
  • Як це виправити? (Потрібно перетворити рядки на числа за допомогою функцій int() або float().)

Математичні операції та функції

Ми перетворили рядок в число. Тепер ми можемо виконувати з ними математичні операції.

Оператори

  • Додавання/Віднімання
  • Множення
  • Звичайне ділення (завжди повертає float)
  • Цілочисельне ділення //

У нас є 140 грн на подарунки. Один подарунок коштує 30 грн. Скільки цілих подарунків ми можемо купити?

total_money = 140
gift_price = 30

# Звичайне ділення дасть нам дріб:
# 140 / 30 -> 4.6666...

# А цілочисельне дасть відповідь:
num_gifts = total_money // gift_price  # -> 4
# Ми можемо купити 4 подарунки.
  • Залишок від ділення (модуль) %

Добре, ми купили 4 подарунки по 30 грн. А скільки решти у нас залишиться?

total_money = 140
gift_price = 30
change = total_money % gift_price  # -> 20
# 140 - (30 * 4) = 20. У нас залишиться 20 грн решти.

Найкращий спосіб перевірити число на парність 

# Якщо залишок від ділення на 2 = 0, число парне
print( 10 % 2 ) # -> 0 (парне)
print( 11 % 2 ) # -> 1 (непарне)
  • Піднесення до степеня **

Ви поклали 1000 грн на депозит під 8% річних. Скільки у вас буде грошей через 5 років?

principal = 1000
rate = 0.08  # 8%
years = 5

final_amount = principal * (1 + rate) ** years
# final_amount -> 1469.328...

Вбудовані функції

  • round()
nasty_number = 33.33333333
pretty_number = round(nasty_number, 2)
  • abs()
price_today = 190
price_yesterday = 210
change = price_today - price_yesterday  # -> -20

# Якщо ми хочемо сказати "Коливання склало 20 грн":
volatility = abs(change)
# volatility -> 20
  • min() / max()
price_from_A = 240.50
price_from_B = 239.99
price_from_C = 241.00

best_price = min(price_from_A, price_from_B, price_from_C)
# best_price -> 239.99

worst_price = max(price_from_A, price_from_B, price_from_C)
# worst_price -> 241.00

Пастка 2: Пастка брудного виводу (Проблема форматування)

  • Окей, ми порахували. Тепер хочемо вивести щось гарне
name = "Alice"
age = 25
print("My name is " + name + " and I am " + age + " years old.")
  • Нам потрібен спосіб цивілізовано вставляти змінні у текст. Для цього існує форматування рядків
name = "Alice"
age = 25
message = f"My name is {name} and I am {age} years old."
print(message)
  • У вашому завданні фінальний рядок дуже складний. Збирати його через + — це жах. F-рядок зробить це завдання тривіальним

Частина 2. Live coding (20 хв))

Завдання:

Напишемо програму, яка питає у користувача ціну товару та відсоток знижки, а потім виводить нову ціну та суму зекономлених коштів

1. Збір вхідних даних

print("--- Калькулятор знижок ---")
# Питаємо ціну. Ціна може бути дробовою (99.99), тому беремо float!
price_str = input("Введіть ціну товару (грн): ")

# Питаємо знижку. Вона теж може бути дробовою (7.5%)
discount_str = input("Введіть відсоток знижки (%): ")

2. Перетворення типів

price = float(price_str)
discount_percent = float(discount_str)

3. Обчислюємо і виводимо результати

discount_amount = price * (discount_percent / 100)

# Рахуємо нову ціну
final_price = price - discount_amount

# Використовуємо f-string для гарного виводу

print(f"Початкова ціна: {price} грн")
print(f"Знижка: {discount_percent}%")
print(f"Ви зекономили: {discount_amount} грн")
print(f"Нова ціна: {final_price} грн")

# Явно округлюємо наші результати
discount_amount_rounded = round(discount_amount, 2)

# Використовуємо :.2f або округлену змінну

print(f"Ви зекономили: {discount_amount_rounded} грн")
print(f"Нова ціна: {final_price:.2f} грн")
